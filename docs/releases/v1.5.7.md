## v1.5.7

### 阅读优先收敛
- 移除偏知识库维护、发布运营和批量修复的重型面板，工具菜单保留阅读、查找、导航、索引诊断、文件信息和导出。
- 删除无入口的旧组件与工具模块，减少启动时可达代码和维护面。
- 欢迎页与 README 重新聚焦本地 Markdown 阅读，不再强调维护仪表盘类流程。

### 启动与大文档性能
- 会话恢复改为懒加载：启动时恢复标签元数据，只读取当前活动文档，其他文件标签在切换到时再读取。
- Markdown 渲染优先使用 Web Worker，解析失败时再回退到主线程解析。
- 启动阶段延后自动更新检查，并避免自动恢复上次文件夹导致索引抢占首屏。
- 新增启动性能压测脚本，默认用 12 个 5MB Markdown 文档模拟大会话。

### 工程与文档
- 新增 `DocumentLoadState`，为待加载、加载中和加载失败的文档提供明确状态。
- `electron:dev` 改为等待 Vite 可用后再启动 Electron，新增 `electron:preview` 和 `perf:startup` 命令。
- 更新 README、docs 首页、AGENTS 指南和发布说明，补齐当前项目结构与发版流程。
- 提供 macOS arm64 与 x64 DMG，自用版为 ad-hoc 签名。
