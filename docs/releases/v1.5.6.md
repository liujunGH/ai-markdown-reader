## v1.5.6

### 阅读工具
- 新增阅读工具侧栏，集中管理高亮、摘录、稍后读、继续阅读、阅读预设、章节进度和分栏阅读。
- 新增批注总览、文档阅读状态卡、章节完成标记和长文章节模式。
- 新增阅读快照，可保存并恢复文档位置、滚动位置、字号、主题和布局。

### 阅读体验
- 新增图片/表格阅读面板，支持图片序列浏览、定位正文、复制图片路径。
- 表格支持全屏阅读、固定表头、复制 Markdown 和 CSV。
- 分屏对比阅读支持同步右侧文档到当前阅读进度。

### 无障碍与恢复
- 新增朗读速度、行距、段落间距、减少动画和高对比高亮设置。
- 阅读时间线支持搜索，并按全部、未读完、已读完筛选。
- 继续阅读、章节进度和阅读会话统计更完整。

### 构建
- 提供 macOS arm64 与 x64 DMG，自用版为 ad-hoc 签名。
