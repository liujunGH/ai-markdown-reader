# 更新日志

## v1.4.0

### 🔒 安全加固
- 新增路径遍历防护：`isPathSafe()` 验证所有文件操作 IPC 路径
- 移除 `app.disableHardwareAcceleration()`，恢复 GPU 硬件加速
- 添加严格 Content-Security-Policy，禁止外部资源加载
- KaTeX CSS 改为本地打包，移除 CDN 依赖
- 禁用 Markdown raw HTML 渲染（`html: false`），收紧 DOMPurify 白名单
- Mermaid securityLevel 升级为 `strict`

### 🚀 性能优化
- Mermaid 按需加载，从 `manualChunks` 中移除预打包
- `parseMarkdown` 使用 `useMemo` 缓存，避免重复渲染
- Prism 语法高亮从 25+ 静态导入精简为 11 个常用语言，支持动态加载
- localStorage 瘦身：文件类标签不再重复存储完整 Markdown 内容

### 🏗️ 架构重构
- 提取 `useKeyboardShortcuts` Hook，`App.tsx` 瘦身 140+ 行
- 新增输入框快捷键拦截：聚焦在 `INPUT`/`TEXTAREA` 内时全局快捷键自动忽略
- 修复 `onOpenFolder` IPC listener 选中子目录的 bug

### 📦 打包与分发
- 支持 Apple Silicon (arm64) 原生构建
- 集成 `electron-updater` 自动更新
- 启用 ASAR 打包，保护源码
- CI 改为多平台矩阵构建（macOS / Ubuntu / Windows）
- 添加 `hardenedRuntime` 和 entitlements 配置框架
